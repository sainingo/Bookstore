## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Have Node installed
- Have Vscode installed

### Setup

### Install
-npm install

### Usage
-npm start

### Run tests
-npm run test

### Deployment
-npm run build
